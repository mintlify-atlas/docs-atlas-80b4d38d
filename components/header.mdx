## Overview

The `Header` component provides a simple, styled header for the CO2 monitoring system. It displays the application title "Sistema de Monitoreo de CO2" and serves as the top-level navigation element.

## Component Location

The component is located at `src/components/Header.js`.

## Implementation

The Header component is a functional React component with minimal complexity:

```javascript
import React from "react";
import './Header.css';

function Header() {
    return (
        <header>
            <h1 className="titulo">Sistema de Monitoreo de CO2</h1>
        </header>
    );
}

export default Header;
```

## Styling

The component uses `Header.css` for styling with responsive design considerations.

### CSS Implementation

```css
.titulo {
    font-size: 2.5em;
    color: #001d1a;
    margin-bottom: 0px;
    padding: 0px;
    font-weight: bold;
}
```

## Props

This component does not accept any props. It renders a static title.

<Info>
  To customize the title text, you would need to modify the component directly or add a `title` prop to the component.
</Info>

## Usage Example

The Header component is used at the top level of the application:

```javascript
import Header from "./components/Header";
import CO2Dashboard from "./components/CO2Dashboard";

function App() {
  return (
    <div>
      <Header />
      <CO2Dashboard />
    </div>
  );
}

export default App;
```

## Customization Options

### Making the Title Dynamic

To make the header accept a custom title prop:

```javascript
function Header({ title = "Sistema de Monitoreo de CO2" }) {
    return (
        <header>
            <h1 className="titulo">{title}</h1>
        </header>
    );
}
```

### Adding Additional Elements

You can extend the header to include navigation or icons:

```javascript
function Header() {
    return (
        <header>
            <h1 className="titulo">Sistema de Monitoreo de CO2</h1>
            <nav>
                {/* Navigation items */}
            </nav>
        </header>
    );
}
```

<Warning>
  When modifying the Header component, ensure you maintain the responsive design by updating the corresponding CSS.
</Warning>
