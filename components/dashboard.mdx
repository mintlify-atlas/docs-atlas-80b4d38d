## Overview

The `CO2Dashboard` component is the core of the CO2 monitoring system. It fetches real-time CO2 data from an API, displays the current reading, and visualizes historical data using an interactive line chart.

<Info>
  The dashboard automatically refreshes every 5 seconds to display the latest CO2 measurements.
</Info>

## Component Structure

The component is located at `src/components/CO2Dashboard.js` and integrates React hooks, Chart.js, and Axios for a complete monitoring solution.

## Time Formatting Function

The `formatTime` function converts timestamps from the API format to a localized time string.

### Implementation

```javascript
const formatTime = (timestamp) => {
    const [datePart, timePartWithPeriod] = timestamp.split(", ");
    const [day, month, year] = datePart.split("/");
    const [timePart, period] = timePartWithPeriod.split(" ");
    let [hours, minutes] = timePart.split(":");
    
    // Convert 12-hour format to 24-hour format
    if (period.toLowerCase() === "p.m." && hours !== "12") {
        hours = parseInt(hours) + 12;
    } else if (period.toLowerCase() === "a.m." && hours === "12") {
        hours = "00";
    }
    
    const isoDate = `20${year}-${month}-${day}T${hours}:${minutes}:00`;
    const date = new Date(isoDate);
    
    if (isNaN(date)) {
        console.error("Fecha no válida:", timestamp);
        return "Invalid Date";
    }
    
    return date.toLocaleTimeString('es-MX', {
        hour: '2-digit',
        minute: '2-digit',
        second: '2-digit',
        hour12: false
    });
};
```

<Note>
  The function expects timestamps in the format: `DD/MM/YY, HH:MM AM/PM` and converts them to 24-hour format.
</Note>

## Data Fetching with useEffect

The component uses React's `useEffect` hook to fetch data at regular intervals.

### Fetch Logic

```javascript
useEffect(() => {
    const fetchData = async () => {
        try {
            const response = await axios.get("https://co2-api.vercel.app/api/co2");
            const { co2, timestamp } = response.data;

            console.log("Nuevo dato recibido:", { timestamp, co2 });

            setLatestCO2(co2);
            setCo2Data((prevData) => {
                const newData = [...prevData, co2];
                if (newData.length > 10) newData.shift();
                return newData;
            });

            setTimestamps((prevTimestamps) => {
                const newTimestamps = [...prevTimestamps, formatTime(timestamp)];
                if (newTimestamps.length > 10) newTimestamps.shift();
                return newTimestamps;
            });
        } catch (error) {
            console.error("Error fetching CO2 data:", error);
        }
    };

    const intervalId = setInterval(fetchData, 5000);
    return () => clearInterval(intervalId);
}, []);
```

<Steps>
  <Step title="API Request">
    Fetches data from `https://co2-api.vercel.app/api/co2` every 5 seconds
  </Step>
  <Step title="Update Latest Reading">
    Sets the `latestCO2` state with the current measurement
  </Step>
  <Step title="Update Historical Data">
    Adds new CO2 value to the array, maintaining only the last 10 readings
  </Step>
  <Step title="Format Timestamps">
    Converts and stores the formatted timestamp for chart labels
  </Step>
</Steps>
