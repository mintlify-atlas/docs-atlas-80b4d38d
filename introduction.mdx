## Welcome to CO2 Monitor

CO2 Monitor is a Progressive Web App (PWA) designed for real-time monitoring of carbon dioxide levels in greenhouse environments. Built with React and Chart.js, it provides interactive visualizations and continuous updates to help you maintain optimal air quality for your crops.

## How It Works

CO2 Monitor fetches real-time CO2 measurements from an external API endpoint and displays the data in an intuitive dashboard. The application:

1. **Polls the API** every 5 seconds to retrieve the latest CO2 readings
2. **Displays current levels** in parts per million (ppm) with large, readable text
3. **Renders a line chart** showing the last 10 measurements for trend analysis
4. **Updates automatically** without requiring manual refresh

The application is built as a PWA, enabling installation on mobile devices for quick access and offline functionality.
