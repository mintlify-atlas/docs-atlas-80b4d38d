## Understanding the Dashboard

The main dashboard component (`src/components/CO2Dashboard.js`) provides real-time monitoring:

```javascript src/components/CO2Dashboard.js
useEffect(() => {
    const fetchData = async () => {
        try {
            const response = await axios.get("https://co2-api.vercel.app/api/co2");
            const { co2, timestamp } = response.data;

            setLatestCO2(co2);
            setCo2Data((prevData) => {
                const newData = [...prevData, co2];
                if (newData.length > 10) newData.shift();
                return newData;
            });

            setTimestamps((prevTimestamps) => {
                const newTimestamps = [...prevTimestamps, formatTime(timestamp)];
                if (newTimestamps.length > 10) newTimestamps.shift();
                return newTimestamps;
            });
        } catch (error) {
            console.error("Error fetching CO2 data:", error);
        }
    };

    const intervalId = setInterval(fetchData, 5000);
    return () => clearInterval(intervalId);
}, []);
```

### Key Features

The dashboard automatically:
- **Fetches data** from the API every 5 seconds (`src/components/CO2Dashboard.js:63`)
- **Maintains history** of the last 10 readings (`src/components/CO2Dashboard.js:49`)
- **Updates the chart** in real-time with formatted timestamps
- **Handles errors** gracefully if the API is unavailable

## First Meaningful Action

Your first interaction with the CO2 Monitor is passive observation. Watch as:

1. **Initial data loads** - The display shows "Cargando..." while fetching the first reading (`src/components/CO2Dashboard.js:83`)
2. **Current reading appears** - A large number displays the latest CO2 level in ppm
3. **Chart populates** - The line graph begins plotting data points as they arrive
4. **Automatic updates** - Every 5 seconds, new data appears without any user action

<Tip>
  Open your browser's developer console (F12) to see real-time logs of incoming data: `"Nuevo dato recibido:"` followed by timestamp and CO2 value (`src/components/CO2Dashboard.js:44`).
</Tip>

## Chart Configuration

The application uses Chart.js with the following registered components (`src/components/CO2Dashboard.js:7`):

```javascript
ChartJS.register(CategoryScale, LinearScale, PointElement, LineElement, Title, Tooltip, Legend);
```

The line chart is configured with:
- **Responsive design** that adapts to screen size
- **Custom tooltips** showing CO2 values in ppm
- **Labeled axes** for time (x-axis) and CO2 levels (y-axis)
- **Color scheme** using teal (#4BC0C0) for visibility
